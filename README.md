# ChatGPT Team 邀请工具使用说明

## 📁 文件位置

```
api_client/
├── invite.py          # 主程序脚本
└── README.md         # 使用说明（本文件）
```

## 🚀 快速开始

### 1. 环境准备

确保你的电脑已安装 Python 3.6 或更高版本。

**检查 Python 版本：**
```bash
python --version
# 或者
python3 --version
```

### 2. 安装依赖

打开命令行/终端，执行以下命令：

```bash
pip install requests
```

**可选：安装彩色输出支持**
```bash
pip install colorama
```

### 3. 下载脚本

将 `invite.py` 文件下载到你的电脑任意位置（比如桌面）。

### 4. 运行脚本

**Windows 用户：**
- 方法1：双击 `invite.py` 文件直接运行
- 方法2：打开命令提示符(CMD)，进入脚本所在目录，运行：
  ```cmd
  python invite.py
  ```

**macOS/Linux 用户：**
```bash
python3 invite.py
```

## 📝 使用步骤

运行脚本后，按照提示依次输入：

### 第 1 步：输入 API 地址

```
请输入API地址
直接回车使用默认地址: http://localhost:58234/api/invite
API地址 >
```

- 如果服务器在本地运行，直接按回车使用默认地址
- 如果服务器在远程，输入完整地址，例如：
  - `http://your-server-ip:58234/api/invite`
  - `https://your-domain.com/api/invite`

### 第 2 步：输入卡密

```
请输入卡密
格式: XXXX-XXXX-XXXX 或 XXXXXXXXXXXX
卡密 >
```

输入你的卡密，支持以下格式：
- `ABCD-1234-EFGH`
- `ABCD1234EFGH`
- `abcd1234efgh`（会自动转大写）

### 第 3 步：输入邮箱

```
请输入接收邀请的邮箱地址
邮箱 >
```

输入要接收 ChatGPT Team 邀请的邮箱地址，例如：
- `user@example.com`
- `zhang.san@gmail.com`

### 第 4 步：确认信息

```
请确认信息
API 地址: http://localhost:58234/api/invite
卡密:     ABCD-1234-EFGH
邮箱:     user@example.com

确认发送邀请？ (y/n) >
```

检查信息无误后，输入 `y` 或 `yes` 确认发送。

### 第 5 步：等待结果

脚本会自动发送请求并显示结果：

**成功示例：**
```
✓ 邀请发送成功！
ℹ 邀请已成功发送！请查收邮箱中的 ChatGPT Team 邀请邮件
```

**失败示例：**
```
✗ 请求参数错误！
✗ 错误信息: 卡密已被使用（使用邮箱: other@example.com）
```

### 第 6 步：继续或退出

```
是否继续发送其他邀请？ (y/n) >
```

- 输入 `y` 继续发送新的邀请
- 输入 `n` 退出程序

## ⚙️ 自定义配置

如果你总是使用同一个服务器地址，可以修改脚本中的默认配置：

1. 用文本编辑器打开 `invite.py`
2. 找到第 30 行：
   ```python
   DEFAULT_API_URL = "http://localhost:58234/api/invite"
   ```
3. 修改为你的服务器地址，例如：
   ```python
   DEFAULT_API_URL = "https://your-domain.com/api/invite"
   ```
4. 保存文件

之后运行时直接按回车即可使用你配置的地址。

## 🐛 常见错误处理

### 错误 1：无法连接到服务器
```
✗ 无法连接到服务器！
可能的原因:
  • API 地址错误
  • 服务器未运行
  • 网络连接问题
```

**解决方法：**
1. 检查 API 地址是否正确
2. 确认服务器是否正在运行
3. 检查网络连接
4. 如果是远程服务器，确认防火墙是否开放端口

### 错误 2：卡密已被使用
```
✗ 错误信息: 卡密已被使用（使用邮箱: xxx@example.com）
```

**解决方法：**
- 每个卡密只能使用一次
- 请使用新的卡密

### 错误 3：卡密不存在
```
✗ 错误信息: 卡密不存在
```

**解决方法：**
- 检查卡密是否输入正确
- 联系管理员确认卡密是否有效

### 错误 4：没有可用的 Team
```
✗ 错误信息: 由于此卡密对应Team账号token配置错误，暂时无法兑换
```

**解决方法：**
- 联系管理员检查后台配置
- 等待管理员修复后重试

### 错误 5：邮箱格式不正确
```
✗ 邮箱格式不正确！
```

**解决方法：**
- 确保邮箱格式正确，包含 `@` 和域名
- 例如：`user@example.com`

## 📞 技术支持

如果遇到问题：
1. 查看上面的错误处理说明
2. 检查服务器日志
3. 联系管理员获取帮助

## 📌 注意事项

1. **卡密只能使用一次**：每个卡密兑换成功后即失效
2. **网络要求**：需要能够访问 API 服务器
3. **Python 版本**：推荐使用 Python 3.6+
4. **邮箱验证**：确保输入正确的邮箱地址

## 🎯 高级用法

### 批量邀请（可选）

如果需要批量发送邀请，可以创建一个批量脚本：

1. 准备一个文本文件 `invites.txt`，每行一个卡密和邮箱（用空格分隔）：
   ```
   ABCD-1234-EFGH user1@example.com
   IJKL-5678-MNOP user2@example.com
   ```

2. 创建批量脚本（需要自己编写）或联系管理员获取批量工具
